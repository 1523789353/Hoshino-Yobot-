1.请安装python安装教程完整安装`python3.8`及以上版本,否则不保证稳定性<br>
&nbsp;&nbsp;&nbsp;可以前往[教程](./Docs/杂交/Install_python3.8.5/Install_Tip.md "傻瓜式安装教程")查看傻瓜式安装教程<br>

&nbsp;&nbsp;&nbsp;本包提供Windows的requirements一键安装部署,请善用`第一次使用.bat`脚本<br>
